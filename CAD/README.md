# Tips for Adaptation:

## Motor
Use any motor you would like (Brushless DC Motors are preferred). Adjust the mount size accordingly.

## CNC Parts
These parts can either be printed OR CNCed. Any thickness is fine.

## Pedals
Pedals can be any width. Adjust the plastic connectors accordingly or use spacers.

## Motor Controllers
You may swap the VESC for a suitable motor controller. Use the [EMC Pro](https://github.com/ebolzMagy/EMCFFBV2) software instead if you plan on doing so (it will be far easier to setup but does cost $10 dollars)

## STM32
Any board that supports USB and UART or CAN will work. If the board supports CAN (F407), use [OpenFFBoard](https://github.com/Ultrawipf/OpenFFBoard).

## Shafts
Use any shaft you'd like (preferably over 8mm). Adjust the holes in the Wheelbase and Pedals accordingly.

### Good Luck and Happy Building!!
